# KlarText

**A plain text editor for macOS.**

*Klar* is German for clear. *Klartext reden* means speaking plainly.

---

## The only format that survived

WordPerfect files from 1985 are unreadable. Word documents from the 1990s are corrupted or stranded in unsupported formats. Every wave of software brought a new file format — and eventually abandoned it.

Plain text is the exception. A `.txt` file written fifty years ago opens today, without conversion, without a subscription, without the original software. It will open fifty years from now.

KlarText is built on that premise. A writing tool that gets out of the way and lets the words last.

---

## What it is

KlarText follows one rule: if a feature is not essential to writing and editing text, it does not belong here.

No syntax highlighting. No plugins. No AI. No distractions. Just text.

---

## Signature features

**Paste and Reflow** `Cmd+Shift+V`
Paste text from PDFs, emails, terminals, or the web. Hard-wrapped lines are rejoined into clean, flowing prose — while lists, code blocks, and shell sessions are left intact.

**Join List as Prose**
Turn a bullet or numbered list into a single flowing paragraph in one step. Each item becomes a sentence: capitalised, punctuated, ready.

**Focus Mode** 
Just text, hidden sidebar, wide margins.

---

## Everything else

- Multi-document sidebar with inline rename and per-tab close
- Regex find & replace via the standard macOS Find bar
- Line numbers, word wrap, show invisible characters
- Automatic encoding detection (UTF-8, UTF-16, Latin-1, Mac Roman)
- Line ending conversion (LF / CRLF / CR) in the status bar
- Go to Line `Cmd+L`, Delete Line `Cmd+Shift+K`
- Block indent / outdent for selections `Tab` / `Shift+Tab`
- System, light, and dark themes
- Native macOS app — not Electron
