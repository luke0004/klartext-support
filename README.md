# KlarText: A Plain Text Editor for macOS

KlarText is a minimalist text editor designed around the principle that plain text is the only format with genuine longevity. As the developer notes, "A `.txt` file written fifty years ago opens today, without conversion, without a subscription, without the original software."

## Core Philosophy

The application adheres to one fundamental principle: features must be essential to writing and editing, or they don't exist in KlarText. This means no syntax highlighting, plugins, AI features, or other distractions—just a focused writing environment.

## Key Capabilities

The editor includes several standout tools:

- **Paste and Reflow** intelligently handles text from various sources, rejoining hard-wrapped lines while preserving lists and code blocks
- **Join List as Prose** converts bullet points into flowing paragraphs with proper capitalization and punctuation
- **Focus Mode** strips away interface elements for distraction-free writing

## Additional Features

Beyond the signature tools, KlarText offers standard text editing functionality including multi-document management, regex find & replace, line numbers, automatic encoding detection, and theme options. It's built as a native macOS application rather than using cross-platform frameworks.
